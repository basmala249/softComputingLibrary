# Soft Computing Library
this library includes techniques like Genetic Algorithm , Fuzzy Systems , Neural network


## Overview
The GeneticAlgorithmTool is a framework designed to implement and experiment with genetic algorithms. It provides a modular structure to define and test various components of genetic algorithms, including chromosomes, selection strategies, crossover strategies, mutation strategies, replacement strategies and fitness functions.

## Components
- **Chromosomes**: Defines different types of chromosomes (Binary, Float, Permutation).
- **ReplacementStrategy**: Implements strategies for replacing populations (SteadyStateReplacement, GeneralReplacement, Elitisim).
- **CrossoverStrategy**: Includes methods like NPointCrossover, OrderCrossover, and UniformCrossover.
- **MutationStrategy**: Provides various mutation techniques (SwapMutation, FloatMutation).
- **SelectionStrategy**: Offers selection methods (RouletteWheelSelection, RankSelection).
- **FitnessFunctions**: Contains functions to evaluate the fitness of chromosomes.
- **CaseStudy**: Includes example application (NQueen).
